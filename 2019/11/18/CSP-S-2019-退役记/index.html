CSP-S 2019 退役记

发表于 2019-11-18 分类于 总结 阅读次数：

比赛之前

测了挺多场多校联测，刚开始两周都认认真真地测多校，大家出的题目也非常正常，都是往年联赛常出的考点，当然，也是我擅长的那些考点。每场测试的成绩马马虎虎，能到十多二十名左右，而hby则似乎没有那么得心应手，这个在之后他做图论相关的题目时让我明确感受到了。之后不知道是哪个学校开的头，大家都开始了“军备竞赛”，好像在比谁出的题能难倒更多的人似的，而且考点一次次地超过联赛，涉及到省选的计数一类我尚未巩固的知识点，于是测起来就艰难阻塞，意兴阑珊。后面大家就各做各题，也没人管多校联测了。

考前一周心里不太放心，特地去做了一下往年的联赛题，写了一下17年的，也是wzy当年的那场。大致算了一下自己赛时的分数，100+60+30+100+100+40，这应该就是我如果在赛场上能拿到的分数了，正好，也是wzy当时的分数，在湖南30+名的样子吧。看完17年的题目之后，本来心慌的我放心了许多，毕竟联赛嘛，还是照顾大多数人的。

要去长沙的前天晚上，老妈还特地带我去逛了一下超市，买了点在路上吃的东西，我还特别买了一瓶桃花酒备用，以防我比赛的时候晚上睡不着觉。

出发了，又是大巴，还是九点，同一个机房，同样的出发地点，只是人变了。不知道下一届的学弟会不会也有这样的感慨呢？想来是不会的吧，毕竟像我这般的闲人属实不多。

路上大家的情绪都蛮欢快，让我感觉全然没有比赛前应该有的紧张，我的心情也被带动着愉悦起来。路上无聊，开始看之前看了前几章的《剑来》，之前看着觉得索然无味，不知所云，这次看了几章之后发现确实比《雪中悍刀行》来得认真多了，毕竟开篇伏笔很多，并不像想到哪写到哪的样子。其实归根结底吸引我的还是他的文笔，不说华丽，却切实古风，不是堆砌辞藻，是真真切切让你感受到作者心里的江湖。比起《雪中悍刀行》中有些情节的不连贯，这本书到目前为止还是没有大的缺憾，这也是我考前收到的惊喜。

到了宾馆，lll和我一个房间，hby则和opf一个房间，虽然摆脱了要喊hby起床之类的繁琐事情，但是晚上的开会在我们房间还是让我心里有点不爽，毕竟我最讨厌麻烦了。

开会大家聊了一下比赛的注意事项，都是zy安排我在赛前说的赛时注意事项里说过的话，炒冷饭罢了，没有很高的兴致，只是点到我的时候我才随口说几句，幸好我是第一个说的，不然等他们说完我可能都不知道说啥了。

会开完我就洗了澡，早早上床睡觉，lll刚躺下没多久，zy就一个电话叫他下去吃烧烤了，最后还是等到他回来之后我才睡着，其实是因为睡着太热，之前一直没找到好办法解决。

第一天

虽然昨晚折腾到十一点左右才睡着，但好在睡眠还是不错，八个小时左右的睡眠，虽然不多，但也绝对不少了。考前还是再看了一下ubuntu下的对拍和背包九讲，顺带也看了看单调队列，总是怕考到DP和计数，心里总是没谱。

进了考场，突然想起自己没带包，还好湖南大学亲切地准备了用来放手机的挂袋，虽然只有我一个人的手机在上面有点尴尬。

开考看题，第一题秒掉了。第二题看了一眼发现只会\(n^2\)DP，写了个50pts就去看T3。T3想了想发现60pts好像可做的样子，链的情况我觉得可以用分治写，不过细节好像比较多，菊花情况每次贪心走一步似乎没啥问题。然后想了想，觉得T2正解应该是个DP，我不太能做得出，T3的60pts部分分虽然难写了点，但好在我会写，然后就果断肝T3了。

最后预计分数100+50+10。\(\text{^_^}\)

心情不是很好，为自己的傻逼感到无语。

离大众分差了50，明天如果题目区分度比较高的话并且我比较擅长应该能追回来吧，毕竟上400+还是有机会的，只要第二天240pts就好了。

晚饭没啥心情吃，随便找了个借口说晚上出来吃烧烤搪塞了过去，就一个人回到房间看看小说。zy在lll之前过来了一趟，直接问我今天怎么回事，估计不知道谁跟他说了吧。我大概说了一下情况，他就安慰我说没啥关系，不要想了，明天好好
